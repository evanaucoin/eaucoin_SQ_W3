# Portal Combat Test

## Concept
This project reimagines a fighting game using the Portal theme. The game takes place in a test chamber where two subjects compete in a combat test.

## Design
The environment uses tiled square panels and a clean grey color palette inspired by the Portal test chambers. Blue and orange colors represent each player, referencing Portal’s visual identity.

## Gameplay
This is a simple fighting game inspired by Street Fighter:
- Player A: A/D to move, W to jump, F to attack
- Player B: Arrow keys to move and jump, K to attack

## Sound
Sound effects were added to improve feedback:
- Start sound when the game begins
- Hit sound when attacks connect
- End sound when the match finishes

## Reflection
I focused on creating a consistent Portal theme across all screens while keeping gameplay simple and responsive.
